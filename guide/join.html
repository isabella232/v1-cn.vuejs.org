社区

参与 Vue 开发

  • 参与规则 - 和其它社区一样，我们有规矩，这是为了让合适的人做合适的事。

  • Vue 的主要组件

    • Vue - 核心库。
    • Vuex - 类 Flux 的应用架构。
    • Vue-router - 单页面应用路由。
    • Vue-resource - 网络请求插件，使用 XMLHttpRequests 或 JSONP 处理响应。
